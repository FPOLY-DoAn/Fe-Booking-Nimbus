# 🌟 Booking Care Nimbus

A modern medical booking platform built with ⚛️ React and ⚡️pnpm — developed by team **Nimbus**.

---

## 📦 Technologies Used

- ✅ React (with Hooks)
- ✅ JavaScript
- ✅ pnpm
- ✅ Rsbuild 
- ✅ MUI
- ✅ RESTful API

---

## 🚀 Getting Started

### Install package:
```bash
pnpm install
```
### Run server
```bash
pnpm dev
```
